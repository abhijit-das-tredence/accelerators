# Accelerators

This repository contains a collection of reusable accelerator packages for multi-cloud and multi-platform data analytics and AI solutions.  
It is organized by technology and platform to make discovery and reuse easy for engineering, delivery, and presales teams.

---

## Repository Structure
/aws → AWS-specific accelerators
/azure → Microsoft Azure accelerators
/databricks → Databricks-focused accelerators
/gcp → Google Cloud Platform accelerators
/multi-cloud → Accelerators that span more than one cloud provider
/snowflake → Snowflake-specific accelerators

## Platform Folders Overview

### **aws/**
Contains accelerator patterns and templates built for the AWS cloud ecosystem.  
Examples may include:
- Data ingestion and transformation pipelines using AWS Glue
- Storage and catalog automation using S3 + IAM

---

### ☁️ **azure/**
Houses accelerators targeting the Azure cloud:
- Azure Data Factory patterns
- Synapse pipelines and automation templates
- Azure DevOps CI/CD best practices

---

### ☁️ **databricks/**
Accelerators focused on the Databricks Lakehouse:
- Bronze → Silver → Gold template pipelines
- Unity Catalog governance patterns
- Medallion framework code examples

---

### ☁️ **gcp/**
Includes accelerators for Google Cloud Platform:
- BigQuery pipeline templates
- Workflow orchestration patterns
- Composer / Dataflow accelerators

---

### ☁️ **multi-cloud/**
Reusable patterns and templates that can be deployed across more than one cloud provider.  
Examples include:
- Multi-cloud monitoring and logging
- Cross-cloud identity and access patterns
- Hybrid data processing pipelines

---

### **snowflake/**
Snowflake-specific accelerators:
- Change management and DDL automation patterns
- Data quality and profiling templates
- Snowflake native performance patterns

---
