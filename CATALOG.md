# Accelerators Catalog

This catalog provides a centralized, searchable inventory of all reusable accelerators in this repository.

It is intended for:
- Delivery teams
- Architects
- Presales engineers
- R&D teams

Use this catalog to quickly find accelerators by **platform, capability, or maturity level**.

---

## How to use this catalog

You can search by:
- Platform → Snowflake, Databricks, AWS, Azure, GCP, Multi-cloud  
- Capability → Ingestion, Governance, Streaming, GenAI, DQ, CI/CD  
- Maturity → PoC, MVP, Production-ready  

Use **Ctrl/Cmd + F** to filter.
